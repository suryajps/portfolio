# Image Folder Structure

Create this folder structure in your portfolio directory:

```
portfolio/
├── images/
│   ├── project1/
│   │   ├── cover.jpg          # Homepage thumbnail (1200x900px recommended)
│   │   ├── image1.jpg         # Full project image
│   │   ├── image2.jpg
│   │   ├── image3.jpg
│   │   ├── image4.jpg
│   │   ├── image5.jpg         # Optional
│   │   └── image6.jpg         # Optional
│   │
│   ├── project2/
│   │   ├── cover.jpg
│   │   ├── image1.jpg
│   │   └── ...
│   │
│   ├── project3/
│   │   └── ...
│   │
│   ├── project4/
│   │   └── ...
│   │
│   ├── project5/
│   │   └── ...
│   │
│   ├── project6/
│   │   └── ...
│   │
│   ├── project7/
│   │   └── ...
│   │
│   └── project8/
│       └── ...
```

## Image Guidelines

**Cover Images (for homepage grid):**
- Aspect ratio: 4:3
- Recommended size: 1200x900px
- Format: JPG or PNG
- File size: Keep under 500KB for fast loading

**Project Detail Images:**
- Width: At least 1600px
- Format: JPG or PNG
- File size: Keep under 1MB each
- Number: 4-6 images per project

## Quick Setup

1. Create the `images` folder in your portfolio directory
2. Inside `images`, create 8 folders: `project1`, `project2`, ... `project8`
3. Add your images to each project folder following the naming convention
4. Update the HTML files with your actual project titles and descriptions
